# 🚀 快速开始

## 最简单的运行方式

### 1. 直接双击打开
直接用浏览器打开 `index.html` 文件即可！

### 2. 命令行启动

```bash
# 使用 Node.js
npm start

# 或使用 Python
python -m http.server 8080

# 然后访问 http://localhost:8080
```

## 🎮 游戏控制

- **WASD** 或 **方向键** - 移动
- 武器会自动攻击
- 拾取经验宝石升级

## 🎯 游戏目标

存活尽可能长的时间！击杀敌人、升级武器、变得越来越强！

## 💡 小技巧

1. 刚开始只有鞭子，要尽快升级获得新武器
2. 优先选择范围和冷却缩减，可以大幅提升输出
3. 注意走位，不要被敌人包围
4. 经验球会自动吸过来，不需要刻意去捡

## 🐛 常见问题

**Q: 游戏运行卡顿？**  
A: 尝试关闭其他标签页，或降低浏览器缩放比例。

**Q: 按键没反应？**  
A: 点击一下游戏画面确保焦点在画布上。
