# v2.8.0 — Onboarding, Replays & Polish

Released: **2026-04-25**

This release is all about making the game friendlier to new players,
funner to share, and tighter under the hood. Still zero runtime
dependencies, **255 tests** green.

## Highlights

### Tutorial

A 5-step interactive tutorial that walks new players through movement,
auto-attacks, picking up XP, levelling up, and pausing. Skippable any
time with **Esc**, and you'll only ever be asked once.

### Replays

Every run is now recorded automatically. The main menu's new
**"Replay Last Run"** button shows the run summary and lets you watch
it back at **1×, 2× or 4×** speed. One slot, ~30 minutes max — your save
won't bloat.

### Mobile Haptics

Phones and tablets now buzz when you take a hit, the boss spawns, you
level up, or you die — four distinct vibration patterns you can
recognise eyes-closed. Toggle in **Settings → Vibration**.

### Customisable Keyboard

Don't like WASD? Open **Settings → Customize controls** and rebind any
of the seven actions (move 4-way, pause, help, mute) to whatever keys
suit you. Conflicts are caught automatically and you can reset to
defaults any time.

### Critical-hit Flash

Brief red screen flash on every crit so you actually feel them land.
Auto-disabled when Reduced Motion is on; manual toggle in Settings.

## Bug Fixes

- Switching language mid-run now retranslates the **entire HUD** (not
  just the boss banner).
- **Speedrun timer** no longer counts time spent paused — your splits
  and final time are now the real running time.
- `localStorage` quota errors no longer spam the console; the game
  silently falls back to in-memory saves.
- Weapons that hit "everything in a radius" (Whip, Bible, Frost Nova,
  Lightning, Mine, explosions) are now **~70% cheaper** on dense waves
  thanks to a spatial-hash query.

## Easter Eggs

The Konami Code still does what you think it does. Find it.

## Upgrade Notes

Existing saves carry over untouched. New settings (`vibration`,
`criticalFlash`, custom keymap) start at sensible defaults. No data
migration required.
